🛡️ PTVoice Pro 极客指南

1. 解禁安全锁 (MacOS 专属)
解压后，不要直接运行 App。右键点击 install.command 选择“用终端打开”，输入开机密码回车。此操作将彻底清除 Apple 隔离属性。
2. 无话筒控制：手机即是指挥棒
电脑无需配置麦克风。 拿起手机微信，使用“语音转文字”发送指令给电脑。 例如说出并发送：pt:全选 或 pt:开始录音。
3. 零延迟执行逻辑
电脑端程序捕获文本后，将瞬间连接 ptvoice.shensist.top 完成 0 元授信，并在本地毫秒级驱动 Pro Tools 执行相应动作。
4. 0 元试用计划
