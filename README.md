## 🚀 软件愿景
PTVoice 是一款专为编曲师设计的 Pro Tools 语音控制终端。通过硬件指纹锁定与云端逻辑对齐，我们为每一位专业用户提供最底层、最直观的交互体验。

## ⚖️ 商业授权说明
1. **单机绑定**：本软件绑定硬件指纹，严禁私自转让或多机共用。
2. **实时同步**：软件运行需保持网络连接，以同步最新的核心编曲逻辑库。
3. **过期提醒**：授权到期前 7 天，系统将通过“灵动之眼”悬浮窗自动弹出预警（黄色状态）。
4. **熔断机制**：到期或违规后，软件将进入锁定模式（红色状态），Pro Tools 物理接口将强制 Mock 化，无法真实控制设备。

## 🛠️ 快速启动
1. 运行 `setup_mac.sh` 进行环境初始化与品牌资产校验。
2. 修改 `config.json` 设定你的 SDK 路径。
3. 启动 `pt_voice.py` 后端服务。
4. 启动 `clipboard_listener.py` 开始语音交互。
